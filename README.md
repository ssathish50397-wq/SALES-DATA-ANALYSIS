**📌 Project Overview**

- This project focuses on analyzing sales data to understand revenue trends, product performance, and customer behavior. The goal is to generate meaningful insights that help businesses improve sales, optimize operations, and make better decisions.

**📁 Dataset Details**

**🔑 Key Features:**

- Product Information (Category, Product Name

- Pricing & Profit Data

-  Customer & Regional Data

- Payment Methods

- Order Details

**🛠️ Tools Used**

- Microsoft Excel

- Pivot Tables

- Pivot Charts
  
- Data Cleaning Techniques
  
**🔄 Workflow**

**1. Data Collection**

- Dataset downloaded from Kaggle

- Imported as CSV file
  
**2. Data Cleaning**

- Removed missing values

- Eliminated duplicate records

- Corrected data types

**3. Data Understanding**

- Defined business goals:

- Increase sales

- Improve customer retention

**Key questions:**

- Which product generates highest revenue?

- Which region performs best?

- How to improve low-performing areas?

**4. Data Transformation**

- Added new columns:

- Sales Performance → Total revenue generated

- Ordered Month → Helps analyze monthly trends

**5. Data Analysis**

- Used Pivot Tables for summarization

- Created Pivot Charts for visualization

- Built dashboards for insights

**📊 Key Insights**

**💰 Profit Analysis**

- Tables & Jackets → Highest profit

- Chairs, Jeans, Sofas → Moderate performance

- Headphones & Tablets → Low profit

**🛒 Sales by Category**

- Furniture → Highest sales

- Clothing → Moderate

- Electronics → Lowest

**⭐ Customer Ratings**

- Average rating: 3 to 3.5

- Indicates moderate customer satisfaction

**💳 Payment Method Analysis**

- UPI & Cash → Most preferred

- Credit Card → Moderate usage

- Debit Card → Least used

**🌍 Region-wise Sales**

- North & West → Top-performing regions

- East → Needs improvement

**🚚 Shipping Cost Analysis**

- Highest: Chennai & Pune

- Moderate: Bangalore, Mumbai, Kolkata

- Lowest: Kochi

**🏙️ City-wise Sales**

- Top Cities: Delhi, Bangalore, Chennai

- Low-performing: Pune, Kochi

**🔍 Final Insights**

- Strong performance in metro cities and major regions

- Furniture category drives maximum revenue

- Digital (UPI) + Traditional (Cash) payments dominate

- High shipping costs may reduce profit margins

- Growth opportunities in weaker regions and cities

**🚀 Conclusion**

- This project demonstrates how Excel, Pivot Tables, and Dashboards can transform raw data into actionable insights, helping businesses improve performance and strategy.

- Dataset sourced from Kaggle.
